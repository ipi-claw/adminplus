## 📝 注意事项

### 后端 API 要求
前端代码已经完成，需要确保后端 API 按照以下规范实现：

1. **统一响应格式**
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

2. **JWT Token 认证**
- 登录接口返回 token
- 所有需要认证的接口需要在请求头中携带：
  ```
  Authorization: Bearer {token}
  ```

3. **分页数据格式**
```json
{
  "records": [...],
  "total": 100,
  "page": 1,
  "size": 10
}
```

4. **API 基础地址**
- 开发环境：`http://localhost:8080`
- 生产环境：`/api`（通过 Nginx 代理）
