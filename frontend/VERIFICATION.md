## 🐛 常见问题

### 1. 登录失败
- 检查后端服务是否启动
- 检查 API 地址配置是否正确
- 检查用户名和密码是否正确

### 2. 页面空白
- 检查浏览器控制台是否有错误
- 检查路由配置是否正确
- 检查组件是否正确导入

### 3. API 请求失败
- 检查后端服务是否启动
- 检查 API 地址配置是否正确
- 检查请求拦截器是否正确配置

### 4. 权限不生效
- 检查用户是否已登录
- 检查权限数据是否正确加载
- 检查权限指令是否正确使用
